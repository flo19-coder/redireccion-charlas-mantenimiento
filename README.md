# Redirección al Registro de Charlas de Mantenimiento

Este repositorio contiene una página estática que redirige automáticamente a:

<https://registro-de-charlas-de-mantenimiento-good-hope-991496397220.us-east1.run.app/>

También muestra un botón para abrir el sitio si el navegador bloquea la redirección automática.

## Publicar con GitHub Pages

En GitHub, entra a **Settings → Pages**, selecciona **Deploy from a branch**, elige la rama `main` y la carpeta `/ (root)`.
